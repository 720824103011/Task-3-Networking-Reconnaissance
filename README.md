# Task 3 - Networking and Reconnaissance

## Objective
To learn basic networking concepts and perform reconnaissance using publicly available information.

## Domain Used
example.com

## Commands Executed

### DNS Lookup
nslookup example.com

### Connectivity Test
ping example.com

### Route Trace
tracert example.com

## Result
Successfully resolved the domain name, verified connectivity, and traced the network path to the destination.

## Conclusion
Basic networking and reconnaissance activities were performed successfully using ethical and publicly available methods.
